##############
think-touchpad
##############

*******************************
ThinkPad TouchPad toggle script
*******************************

:Author: Martin Ueding <dev@martin-ueding.de>
:Date: 2012-06-08
:Manual section: 1

SYNOPSIS
========

::

    think-touchpad

DESCRIPTION
===========

This scripts toggles the TrackPad. It is designed to work on ThinkPads, but it
will probably work on almost all laptops.
